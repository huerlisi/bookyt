h1. mailyt

mailyt is a simple but powerfull financial accounting software.

h2. About mailyt

mailyt is a web application implementing double-entry bookkeeping providing:

* Customizable accounts
* Booking wizard
* Booking templates
* Earnings report
* Balance sheet

mailyt is built on "Ruby On Rails":http://www.rubyonrails.org and released as "Open Source":http://www.opensource.org.
